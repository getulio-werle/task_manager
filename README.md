# Task Manager

## Descrição
Aplicação web para gerenciamento de tarefas. Foi desenvolvida durante um curso de Desenvolvimento Web para conhecer o essencial do framework Laravel.

## Tecnologias utilizadas
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- Laravel
- Biblioteca DataTables
